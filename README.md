# 📦 Gen-Z Inventory & Billing System

A modern, full-stack, web-based inventory management and billing system built using Python's Flask framework. Features a stunning, responsive front-end interface with light/dark mode and smooth animations.

## 🌟 Key Features

- **Inventory Management:** Full CRUD operations for products. Easily add, edit, and categorize inventory items.
- **Smart Billing & Invoicing:** Streamline your checkout process and generate invoices.
- **Dashboard Analytics:** Visual top-selling products and sales velocity metrics so you know how your store is doing.
- **Modern UI/UX:** A robust "Gen-Z" aesthetic design boasting dynamic light/dark modes, sleek micro-animations, and a fully responsive layout.
- **Secure Authentication:** User login and management powered by Flask-Login.

## 🛠️ Technology Stack

- **Backend:** Python, Flask 
- **Database:** SQLite (powered by Flask-SQLAlchemy)
- **Frontend:** Vanilla HTML, CSS, JavaScript
- **Security:** Werkzeug for strong password hashing
